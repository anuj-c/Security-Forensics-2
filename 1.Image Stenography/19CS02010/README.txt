To encode the image run the encode.py file with the following command:
python3 encode.py

Enter the key (preferably more than 6 digits).
Enter the message to hide in the image.

After successfully encoding the image, the encoded image will be saved in the same directory as res.png.

To decode the image run the decode.py file with the following command:
python3 decode.py

Enter the key which was used while encoding.
If the key is correct you will see the secret message.
